# BOLGLONG
简易博客社区系统开发

工具：Eclipse。
数据库为MySQL，后端使用servlet，JDK最好使用1.8，tomcat为8.5。

该项目一个简单博客社区实现。
目前包含的基本功能有：
- 登录/注册
- 用户页面
- 评论功能
- 文章分类
- 文章管理(添加/修改/删除)
- 写文(使用了editor.md)
